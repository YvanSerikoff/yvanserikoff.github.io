Description

This project involved creating a game in JAVA using agile methodology. We were a randomly selected group of 5. As the aim of the project was to educate us in agile methodology, we were accompanied by professional SCRUM Masters to help us get organized. Learning and putting this methodology into practice was the main difficulty of the project (what's more, with “strangers”), but thanks to our different JAVA skills, the organization of the work came naturally.

Skills

This project taught us to work alongside a big team using the agile methodology and has strengthened our JAVA programming skills.
