Description

This project involved the development of a social network in JEE, enabling several users to communicate with each other via Threads. Each user can create threads, post messages on them and interact with other's ones with likes. This being my first programming project on my own, I had to learn to be autonomous in my work as well as curious in my research to be able to deliver a project that suited me. I also had to learn JEE programming and the MVC Web model.
