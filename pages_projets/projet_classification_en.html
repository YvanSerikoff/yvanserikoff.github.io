Description

This project involved developing a dataset classification application using a KNN algorithm. The user can enter a dataset and then visualize it, change the axes, and add a point that will then be classified using the KNN algorithm. It was to be completed in groups of four, and organizing a project of this scale was undoubtedly the most complicated task.

Skills

This project taught us the importance of organization and rigor in group work.
