Semestre 5 Flutter Kotlin

Projet Erasmus

Une application mobile conçue en équipe pendant un stage à Coimbra, avec un vrai contexte client, des retours réguliers et des responsabilités techniques concrètes.

Description

Application de maintenance permettant de suivre les tâches assignées, mettre à jour leur statut, et historiser les interventions. Intégration d’une API REST et gestion de l’authentification ainsi que d'un système de visualisation 3D des éléments de maintenance.

Côté mobile : Flutter pour l’UI multi-plateformes. Côté Android natif : modules spécifiques en Kotlin pour les fonctionnalités de réalité augmentée.
