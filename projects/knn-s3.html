Semestre 3 JavaFX Machine Learning

Projet classification S3 (KNN)

Une première mise en pratique de la classification supervisée, avec une interface simple pour charger les données, paramétrer K et visualiser le résultat.

Description

Implémentation d’un classifieur K-Nearest Neighbors sur un jeu de données d’exemple, avec interface JavaFX pour charger des données, ajuster K, et visualiser les résultats.

Travail sur la normalisation des features, la distance (euclidienne/manhattan), et la validation croisée basique.
